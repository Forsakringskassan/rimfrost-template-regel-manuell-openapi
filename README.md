# rimfrost-template-regel-manuell-openapi

Detta repository är en mall för OpenAPI-specifikationer för manuella regler.

## Syfte

Mallen används som utgångspunkt när nya API-specifikationer ska skapas för manuella regler. Den används även av mallen för manuella regler för att generera API-klienter och serverklasser. Den innehåller en grundläggande struktur och exempel på vanliga scheman och endpoints.

## Innehåll

- `openapi.yaml` - OpenAPI 3.0.3-specifikation med exempel på endpoints och scheman

## Användning

1. Skapa ett nytt repository baserat på denna mall
2. Anpassa `openapi.yaml` efter den specifika regelns behov
3. Uppdatera scheman och endpoints enligt kraven för den manuella regeln
